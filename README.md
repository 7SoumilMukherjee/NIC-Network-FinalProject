# NIC-Network-FinalProject
Simulation of a government-scale NIC network architecture in Cisco Packet Tracer featuring eBGP-based inter-Super PoP connectivity, OSPF-enabled regional networks, route redistribution, VLAN segmentation, inter-VLAN routing, DHCP services, and redundant high-availability network design.
# Cisco Network Design Project

## Objective
Designed and simulated an enterprise network using Cisco Packet Tracer.

## Features
- VLAN configuration
- Static routing
- DHCP
- NAT
- Access Control Lists

## Tools Used
- Cisco Packet Tracer

## Files
- network-topology.pkt
- topology.png

## How to Open
1. Download the .pkt file.
2. Open it using Cisco Packet Tracer.
